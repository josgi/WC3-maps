Description

Scourge Campaign: Legacy of the Damned
Co-op Mode

Turnro and Ljarvisone present for the first time the original Scourge Campaign in Co-op mode.
Now you can enjoy this classic campaign with a friend as you relive the story of Arthas' ascension to the Frozen Throne and becoming the Lich King.





Features and Changes Implemented

Modified the Scourge Campaign for 2 players, complete with save/load Hero functionality between maps

Created Anub'arak-only scene in Chapter 7 Part Three, complete with new terrain and events

Included 2 new Heroes: Falric in Chapter 2 and Sharlindra in Chapter 3. These heroes come with their own models and abilities

Heroes from Chapter 2 on-wards possess a sachet to carry additional items for later use

6 new legendary items to find throughout the campaign

Added QWER hotkeys for all units, buildings, spells, etc

Made Normal and Hard difficulties slightly harder to cater for 2 players

New models to runes, potions, and other commonly used consumable items

Screenshots and Legendary Items


Cultist Ring
Increases the Hero's hit point regeneration by 2 hit points per second and mana regeneration by 50%
Voodoo Mask
Increases the Agility of the Hero by 10 when worn. Also increases nearby ranged units' damage by 10%
Scourge Shoulder Plates
Increases the armor of the Hero by 4 when worn. Also grants friendly nearby melee units life stealing attacks which take 15% of the damage they deal and convert it into life
Ancient Amulet
Grants the ability to fire bolts of pain that deal 250 damage. Also increases the Hero's hit points by 300 when worn
Nerubian Scepter
Summons 2 Nerubian Spiderlings to fight for you. Also increases the Intelligence of the Hero by 12 when worn
Crown of the Forgotten King
Increases the Strength of the Hero by 12 when worn. Also reduces Magic damage taken by 33%



Help and Resources

Please refer to the answers below before posting questions.


1. Download the campaign ZIP file onto your computer. (DOWNLOAD button near the top of the page)
2. Unzip the file and extract its contents into "Maps/Download" in your Warcraft 3 folder.

Unzipping files can vary depending on your operating system (Windows, Mac, etc), but it should involve something like this:
1. Right click on .zip file that you want to unzip and click on "Extract All".
2. In the "Extract Compressed (Zipped) Folders" dialog, enter a folder path where you want files to be extracted.Once done, click on "Extract" button to start the extraction.

Once the map files have been extracted into your Warcraft 3 folder, you will be able to play the Scourge Campaign as a Custom Game for each individual map. (Battle-net play is currently being investigated)



How does it work?
Each time you complete a mission, a Hero save code is automatically saved in a text file on both player's computers. At the beginning of the next map, this code can be used to recover your Heroes by doing the following (only one player needs to do this):
a. Minimize the current game (Alt+Tab).
b. Open "CustomMapData\{CHAPTER NAME}.txt" in your Warcraft III folder.
c. Copy (Ctrl+C) the "load xxxx-xx..." text in the text file. Do not close the file, as doing so will make the code not copy over prior to Warcraft 3 versions 1.28.
d. Go back into Warcraft 3 and paste (Ctrl+V) the code into chat and press ENTER.

Which player will have the save file?
The text file containing the Hero save code will save on both player's computers unless one player dropped out during the mission.

Where is my Warcraft 3 folder?
This will depend on your version of Warcraft 3:
Prior to 1.28 -> Either on your C drive or the location in which you installed Warcraft 3. An example path could be "C/Program Files/Warcraft 3"
1.28 and beyond -> Located in your Documents folder

Copying the text code into Warcraft 3 chat doesn't work. What should I do?
If the code does not appear in chat after pressing Ctrl+V, this is because you closed the text file before pasting the code into Warcraft 3.
If the code you copied into Warcraft 3 results in an error message, this could be either the code was modified or you are attempting to load the save code for the wrong map.
If the code you copied and entered into chat does not do anything, this is most likely a bug and should be reported to either Turnro or Ljarvisone.



- Maps have been worked on while keeping true to their original versions
- Improved save/load system. Players no longer have to remember and type long tedious codes! Instead, the game will automatically store codes in a text file that can be copied and pasted straight into chat
- Players get one main hero in each mission, with bases automatically shared between them. Earlier 2-player campaigns had players control separate bases, requiring huge balance changes and altering the original feel of the maps
- Maps can be played solo, especially if one player drops out of the game. This gives the other player the choice of either leaving or continue on to win the mission for the team
- New heroes fit the story of the campaign and do not feel poorly added
